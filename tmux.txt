START NEW 
        tmux

START NEW SESSION
        tmux new -s <session_name>

ATTACH RECENT SESSION
        tmux a 

ATTACH TO NAMED SESSION
        tmux a -t <session_name>

LIST SESSION
        tmux ls 

KILL SESSION 
        tmux kill-sessions -t <session_name> 

SHORTCUTS
        ctr + b 
                s  list sessions
                $  name session
                c  create window
                w  list windows
                n  next window
                p  previous window
                f  find window
                ,  name window
                &  kill windowa

                %  vertical split
                "  horizontal split
                o  swap panes
                q  show pane numbers
                x  kill pane
                +  break pane into window (e.g. to select text by mouse to copy)
                -  restore pane from window
                space - toggle between layouts
                { (Move the current pane left)
                } (Move the current pane right)
                z toggle pane zoom
                

                d  detach
                t  big clock
 
                ?  list shortcuts
                :  prompt
